GitHub+Hexo搭建个人网站

发表于 2020-02-14 分类于 Hexo 阅读次数： Valine：
本文字数： 2.9k 阅读时长 ≈ 3 分钟

前言：

本篇介绍基于GitHub+Hexo搭建一个免费的、无限流量的个人网站，通过网络任何人可以访问你的网站。以下在虚拟机的Ubuntu18.04 (Ubuntu 16.04也可以)上安装。以root用户登录或普通用户登录后获取root权限。

什么是Hexo？Hexo是一个快速、简洁且高效的博客框架。基于Node.js，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub上。Hexo同时也是GitHub上的开源项目，因为Hexo的创建者是台湾人，对中文的支持很友好，可以选择中文进行查看。

搭建步骤：

  1. 创建GitHub个人仓库
  2. 安装Git
  3. 安装Node.js
  4. 安装Hexo
  5. 将Hexo部署到GitHub上

1. 创建GitHub个人仓库

登陆到GitHub，如果没有GitHub账号，使用你的邮箱注册GitHub账号，注册完之后点击New repository创建新仓库，仓库名格式应该为：用户名.github.io。这个用户名使用你的GitHub注册的用户名。

2. 安装Git

什么是Git？简单来说Git是开源的分布式版本控制系统，可以有效的进行项目版本管理。在这里是用来管理你的Hexo博客文章，上传到GitHub的工具。

添加ssh key到GitHub

ssh key是一个认证，让github识别绑定这台机器，允许这台机器提交。

(1) 配置生成ssh key

然后输入：

Generating public/private rsa key pair.
Enter file in which to save the key(/root/.ssh/id_rsa): #直接按回车
Created directory '/root/.ssh'.
Enter passphrase(empty for no passphrase): #直接按回车
Enter same passphrase again: #直接按回车

这个命令将会生成一个以你邮箱为标题的ssh key，其存放路径为~/.ssh/下。

(2) 复制ssh key到GitHub

在~/.ssh/文件夹下，查看里面有id_rsa和id_rsa.pub这两个文件。

登陆GitHub系统，点击Settings->SSH and GPG keys->New SSH key,Title自定义，复制id_rsa.pub的公钥内容到Key中，最后点击Add SSH key。

3. 安装Node.js

用apt源安装的node版本低，从https://nodejs.org/dist/v12.15.0/ 这里下载node-v12.15.0-linux-x64.tar.xz，以二进制压缩包方式安装。

(1) 解压二进制安装包到你想安装的路径

我使用/usr/local/lib/nodejs

(2) 设置环境变量~/.profile，添加以下环境变量到底部

# Nodejs

VERSION=v12.15.0
DISTRO=linux-x64
export PATH=/usr/local/lib/nodejs/node-$VERSION-$DISTRO/bin:$PATH
